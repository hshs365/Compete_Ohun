# 카카오맵 스크립트 로드 실패 해결 가이드

## ❌ 현재 발생 중인 에러

```
카카오맵 스크립트 로드 실패
요청 URL: https://dapi.kakao.com/v2/maps/sdk.js?appkey=966de50...
```

## 🔍 원인 확인 방법

### 1단계: Network 탭에서 확인 (가장 중요!)

1. **F12 → Network 탭 클릭**
2. **필터에 "kakao" 입력**
3. **`dapi.kakao.com/v2/maps/sdk.js` 요청 찾기**
4. **상태 코드 확인**:
   - ✅ **200 OK**: 정상 (다른 문제일 수 있음)
   - ❌ **403 Forbidden**: JavaScript 키 또는 플랫폼 설정 문제
   - ❌ **404 Not Found**: URL 문제 또는 키 문제
   - ❌ **CORS 에러**: 플랫폼 설정에 localhost:5173이 없음

### 2단계: 새 앱 설정 확인

새로운 앱으로 전환했으므로 다음을 확인해야 합니다:

#### A. JavaScript SDK 도메인 등록 확인
1. **"앱 설정" > "앱" > "플랫폼 키"** 페이지로 이동
2. **"JavaScript 키"** 섹션에서
3. **"JavaScript SDK 도메인"**에 `http://localhost:5173`이 등록되어 있는지 확인
4. **없으면 다시 등록**

#### B. 추가 기능 신청 확인
1. **"앱 설정" > "앱" > "추가 기능 신청"** 페이지로 이동
2. **카카오맵**이 승인되었는지 확인
3. **승인되지 않았다면 다시 신청**

#### C. 카카오맵 활성화 확인
1. **"제품 설정" > "카카오맵"** 페이지로 이동
2. **"사용 설정" > "상태"** 토글이 ON인지 확인
3. **OFF라면 ON으로 변경**

## 🔧 해결 방법

### 방법 1: JavaScript SDK 도메인 재등록

1. **"앱 설정" > "앱" > "플랫폼 키"** 페이지로 이동
2. **"JavaScript 키"** 섹션에서
3. **"JavaScript SDK 도메인"** 섹션 찾기
4. **`http://localhost:5173` 입력**
5. **저장**

### 방법 2: 추가 기능 신청

1. **"앱 설정" > "앱" > "추가 기능 신청"** 페이지로 이동
2. **카카오맵 찾기**
3. **"신청" 또는 "요청" 버튼 클릭**
4. **승인 대기**

### 방법 3: 카카오맵 활성화

1. **"제품 설정" > "카카오맵"** 페이지로 이동
2. **"사용 설정"** 섹션에서
3. **"상태" 토글을 ON으로 변경**
4. **저장**

## ✅ 체크리스트

### 필수 확인 사항:
- [ ] **Network 탭에서 상태 코드 확인** (403, 404, CORS 등)
- [ ] **JavaScript SDK 도메인**: `http://localhost:5173` 등록 확인
- [ ] **추가 기능 신청**: 카카오맵 승인 확인
- [ ] **카카오맵 활성화**: 상태 ON 확인
- [ ] **클라이언트 재시작**: `.env` 파일 변경 후 재시작했는지 확인

1. **Network 탭에서 상태 코드 확인**
   - 가장 먼저 확인해야 할 사항

   - 새 앱으로 전환했으므로 다시 신청 필요할 수 있음

4. **카카오맵 활성화 확인**
   - 새 앱으로 전환했으므로 다시 활성화 필요할 수 있음

5. **브라우저 완전히 닫고 다시 열기**

   - `.env` 파일 변경 후 재시작했는지 확인

- **추가 기능 신청**도 새 앱에서 다시 신청해야 할 수 있습니다

## 🆘 Network 탭 상태 코드별 해결 방법

### 403 Forbidden:
- JavaScript 키가 잘못되었거나
- 플랫폼 설정에 localhost:5173이 없거나
- 추가 기능 신청이 승인되지 않았거나
- 카카오맵이 활성화되지 않음

### 404 Not Found:
- JavaScript 키가 잘못되었거나
- URL이 잘못됨

### CORS 에러:
- 플랫폼 설정에 localhost:5173이 없음
- JavaScript SDK 도메인이 등록되지 않음

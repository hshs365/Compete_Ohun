# JavaScript SDK 도메인 설정 대안 방법

## 🔍 JavaScript SDK 도메인을 찾을 수 없는 경우

플랫폼 키 페이지에서 "JavaScript SDK 도메인" 섹션이 보이지 않는다면, 다음 방법들을 시도해보세요:

## 방법 1: JavaScript 키 카드 클릭/확장

1. **"대표 Default JS Key" 카드 클릭**
   - 카드를 클릭하면 상세 설정이 펼쳐질 수 있음
   - 또는 카드 오른쪽의 "⋮" (세 점) 메뉴 클릭

2. **"수정" 또는 "설정" 옵션 찾기**
   - 카드를 클릭하면 모달이나 상세 페이지가 열릴 수 있음
   - 그 안에 "JavaScript SDK 도메인" 설정이 있을 수 있음

## 방법 2: 플랫폼 설정 페이지 확인

실제로는 **"플랫폼" 설정 페이지**에서 Web 플랫폼을 등록하는 것이 JavaScript SDK 도메인 설정일 수 있습니다.

### 사이드바 경로:
```
앱 설정
  └─ 앱
      └─ 플랫폼 (Platform)
```

**단계:**
1. **"앱 설정" > "앱" > "플랫폼" 클릭**
2. **"Web 플랫폼 등록" 버튼 클릭**
3. **사이트 도메인 입력**: `http://localhost:5173`
4. **저장**

### 또는 URL로 직접 접속:
```
https://developers.kakao.com/console/app/1358790/config/platform
```

## 방법 3: 카카오맵 API 가이드 재확인

카카오맵 API 가이드에서 확인한 내용:
- "JavaScript SDK 도메인"은 플랫폼 키의 JavaScript 키에서 설정
- 하지만 실제 UI에서는 "플랫폼" 설정의 Web 플랫폼 등록과 동일할 수 있음

## ✅ 권장 방법

**"플랫폼" 설정 페이지에서 Web 플랫폼 등록을 시도해보세요:**

1. **"앱 설정" > "앱" > "플랫폼" 클릭**
2. **"Web 플랫폼 등록" 클릭**
3. **`http://localhost:5173` 입력**
4. **저장**

이것이 JavaScript SDK 도메인 설정과 동일한 기능일 가능성이 높습니다.

## 🧪 테스트

설정 후:
1. **브라우저 완전히 닫기**
2. **클라이언트 재시작**
3. **브라우저 다시 열기**
4. **`http://localhost:5173` 접속**
5. **F12 → Network 탭**
6. **`dapi.kakao.com/v2/maps/sdk.js` 요청 확인**
7. **상태 코드가 200 OK인지 확인**

## 📝 참고

카카오 개발자 콘솔의 UI가 변경되었을 수 있습니다:
- 이전: "JavaScript SDK 도메인" 별도 설정
- 현재: "플랫폼" 설정의 Web 플랫폼 등록과 통합

**"플랫폼" 설정에서 Web 플랫폼을 등록하는 것이 올바른 방법일 가능성이 높습니다!**

# 카카오 개발자 콘솔 플랫폼 설정 가이드

## 📍 플랫폼 설정 위치

### 사이드바 경로:
```
앱 설정
  └─ 앱
      └─ 플랫폼 (Platform)
```

**단계별 안내:**

1. **왼쪽 사이드바에서 "앱 설정" 클릭**
2. **"앱" 섹션 확장** (화살표 클릭)
3. **"플랫폼" 메뉴 클릭**
   - "플랫폼 키" 아래에 있을 수 있음
   - 또는 "일반" 옆에 있을 수 있음

### 만약 "플랫폼" 메뉴가 보이지 않는 경우:

1. **"앱 설정" > "앱" > "일반" 클릭**
2. 페이지에서 "플랫폼" 탭 찾기
3. 또는 URL로 직접 접속:
   ```
   https://developers.kakao.com/console/app/1358790/config/platform
   ```

## 🌐 Web 플랫폼 등록 방법

1. **"플랫폼" 페이지 접속**
2. **"Web 플랫폼 등록" 버튼 클릭**
3. **사이트 도메인 입력**: `http://localhost:5173`
4. **저장** 클릭
5. **1-2분 대기** (변경 사항 반영 시간)

## 🗺️ 카카오맵 사용 설정

### 현재 상태 확인:
- **제품 설정** > **카카오맵** 페이지에서
- **"사용 설정"** 섹션의 **"상태"** 토글이 **OFF**로 되어 있음

### 카카오맵 활성화 방법:

#### 방법 1: 비즈앱 전환 없이 사용 (일반 개발자)

1. **"제품 설정" > "카카오맵" 페이지 접속**
2. **"사용 설정"** 섹션에서
3. **"상태" 토글을 ON으로 변경**
4. **저장**

**참고**: 
- 일반 개발자도 카카오맵 JavaScript API는 사용 가능합니다
- REST API (로컬 API)는 별도 활성화가 필요할 수 있습니다

#### 방법 2: 로컬 API 활성화 (주소 변환용)

1. **"제품 설정"** 메뉴에서
2. **"로컬"** 찾기 (카카오맵과 별도 메뉴일 수 있음)
3. **"로컬" 활성화** 클릭
4. **저장**

**참고**:
- 로컬 API는 주소 → 좌표 변환에 사용됩니다
- JavaScript API와는 별도로 활성화해야 합니다

## ✅ 체크리스트

### 필수 설정:
- [ ] **플랫폼 설정**: Web 플랫폼에 `http://localhost:5173` 등록
- [ ] **카카오맵 사용 설정**: 상태를 ON으로 변경
- [ ] **로컬 API 활성화**: 주소 변환 기능 사용 시 필요

### 확인 방법:
1. **플랫폼 설정 확인**:
   - "앱 설정" > "앱" > "플랫폼"에서 Web 플랫폼 목록 확인
   - `http://localhost:5173`이 등록되어 있어야 함

2. **카카오맵 설정 확인**:
   - "제품 설정" > "카카오맵"에서 상태가 ON인지 확인

3. **로컬 API 확인**:
   - "제품 설정"에서 "로컬" 메뉴 찾기
   - 활성화되어 있는지 확인

## 🆘 문제 해결

### "플랫폼" 메뉴를 찾을 수 없는 경우:
- URL로 직접 접속 시도
- 브라우저 새로고침
- 다른 브라우저에서 시도

### 카카오맵 상태를 ON으로 변경할 수 없는 경우:
- 페이지 새로고침 후 다시 시도
- 다른 제품 설정(카카오 로그인 등)이 활성화되어 있는지 확인

# 시크릿 모드에서도 에러 발생 시 해결 가이드

## ❌ 현재 상황

시크릿 모드에서도 `net::ERR_BLOCKED_BY_CLIENT` 또는 스크립트 로드 실패가 발생합니다.

이것은 **확장 프로그램 문제가 아니라 카카오 개발자 콘솔 설정 문제**입니다.

## 🔍 확인해야 할 사항

새로운 앱으로 전환했으므로 **모든 설정을 다시 확인**해야 합니다.

### 1단계: JavaScript SDK 도메인 확인 (가장 중요!)

1. **카카오 개발자 콘솔 접속**
   - `https://developers.kakao.com/console/app/1359677/config/platform-key`
   - (앱 ID가 1359677로 변경되었으므로 URL도 변경됨)

2. **"JavaScript 키" 섹션 찾기**
   - "대표 Default JS Key" 카드 확인
   - 키 값: `966de50d9809428a90270aa613b8619a`

3. **"JavaScript SDK 도메인" 섹션 확인**
   - JavaScript 키 카드를 클릭하거나
   - "수정" 버튼 클릭
   - "JavaScript SDK 도메인" 섹션에서
   - **`http://localhost:5173`이 등록되어 있는지 확인**
   - **없으면 다시 등록**

### 2단계: 추가 기능 신청 확인

1. **"앱 설정" > "앱" > "추가 기능 신청"** 페이지로 이동
   - URL: `https://developers.kakao.com/console/app/1359677/config/feature`

2. **카카오맵 찾기**
   - 카카오맵이 목록에 있는지 확인
   - 상태가 "승인됨" 또는 "활성화됨"인지 확인

3. **승인되지 않았다면**
   - "신청" 또는 "요청" 버튼 클릭
   - 승인 대기

### 3단계: 카카오맵 활성화 확인

1. **"제품 설정" > "카카오맵"** 페이지로 이동
   - URL: `https://developers.kakao.com/console/app/1359677/product/kakao-map`

2. **"사용 설정" 섹션 확인**
   - "상태" 토글이 **ON**인지 확인
   - OFF라면 ON으로 변경
   - 저장

### 4단계: 로컬 API 활성화 확인 (주소 변환 기능 사용 시)

1. **"제품 설정"** 메뉴에서
2. **"로컬"** 찾기
3. **활성화되어 있는지 확인**
4. 활성화되지 않았다면 활성화

## 🔧 해결 방법

### 방법 1: JavaScript SDK 도메인 재등록

1. **"앱 설정" > "앱" > "플랫폼 키"** 페이지로 이동
2. **"JavaScript 키"** 섹션에서
3. **"대표 Default JS Key" 카드 클릭** 또는 **"수정" 버튼 클릭**
4. **"JavaScript SDK 도메인" 섹션 찾기**
5. **`http://localhost:5173` 입력**
6. **저장**

### 방법 2: 추가 기능 신청

1. **"앱 설정" > "앱" > "추가 기능 신청"** 페이지로 이동
2. **카카오맵 찾기**
3. **"신청" 또는 "요청" 버튼 클릭**
4. **승인 대기** (보통 몇 분 ~ 몇 시간)

### 방법 3: 카카오맵 활성화

1. **"제품 설정" > "카카오맵"** 페이지로 이동
2. **"사용 설정"** 섹션에서
3. **"상태" 토글을 ON으로 변경**
4. **저장**

## ✅ 체크리스트

### 필수 확인 사항:
- [ ] **JavaScript SDK 도메인**: `http://localhost:5173` 등록 확인
- [ ] **추가 기능 신청**: 카카오맵 승인 확인
- [ ] **카카오맵 활성화**: 상태 ON 확인
- [ ] **로컬 API 활성화**: 주소 변환 기능 사용 시 필요

## 🧪 테스트

### 설정 완료 후:

1. **브라우저 완전히 닫기**
2. **클라이언트 재시작**
   ```bash
   cd client
   npm run dev
   ```
3. **브라우저 다시 열기**
4. **`http://localhost:5173` 접속**
5. **F12 → Network 탭**
6. **필터에 "kakao" 입력**
7. **`sdk.js` 요청 확인**
8. **상태 코드가 200 OK인지 확인**

## 📝 중요 사항

새로운 앱으로 전환했으므로:
- **모든 설정을 처음부터 다시 해야 합니다**
- **JavaScript SDK 도메인**은 새 앱에서 다시 등록해야 합니다
- **추가 기능 신청**도 새 앱에서 다시 신청해야 합니다
- **카카오맵 활성화**도 새 앱에서 다시 활성화해야 합니다

## 🆘 여전히 문제가 있는 경우

1. **Network 탭에서 실제 상태 코드 확인**
   - 403 Forbidden: JavaScript 키 또는 플랫폼 설정 문제
   - 404 Not Found: URL 문제 또는 키 문제
   - CORS 에러: JavaScript SDK 도메인 미등록

2. **카카오 개발자 센터에 문의**
   - 모든 설정을 확인했는데도 작동하지 않는다면

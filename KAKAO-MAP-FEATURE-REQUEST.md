# 카카오맵 추가 기능 신청 가이드

## ❌ 현재 상황

카카오맵 상태 토글을 ON으로 변경하려고 하면 다음 팝업이 나타납니다:
```
카카오맵 권한이 없습니다.
[앱 설정] > [앱] > [추가 기능 신청]에서 요청할 수 있습니다.
```

## 📍 추가 기능 신청 위치

### 사이드바 경로:
```
앱 설정
  └─ 앱
      └─ 추가 기능 신청 (Request Additional Features)
```

**단계별 안내:**

1. **왼쪽 사이드바에서 "앱 설정" 클릭**
2. **"앱" 섹션 확장** (화살표 클릭)
3. **"추가 기능 신청" 메뉴 클릭**
   - "플랫폼 키" 아래에 있을 수 있음
   - 또는 "일반" 옆에 있을 수 있음

### 또는 URL로 직접 접속:
```
https://developers.kakao.com/console/app/1358790/config/feature
```

## 🔧 카카오맵 추가 기능 신청 방법

### 1단계: 추가 기능 신청 페이지 접속

1. **"앱 설정" > "앱" > "추가 기능 신청" 클릭**
2. 또는 위 URL로 직접 접속

### 2단계: 카카오맵 신청

1. **"카카오맵" 찾기**
   - 목록에서 "카카오맵" 또는 "Kakao Map" 찾기
2. **"신청" 또는 "요청" 버튼 클릭**
3. **신청 사유 입력** (필요한 경우):
   - 예: "개인 프로젝트 개발용", "지도 기능 테스트" 등
4. **제출**

### 3단계: 승인 대기

- **승인 시간**: 보통 몇 분 ~ 몇 시간 소요
- **알림 확인**: 승인 완료 시 알림이 올 수 있음
- **상태 확인**: "추가 기능 신청" 페이지에서 승인 상태 확인

### 4단계: 카카오맵 활성화

1. 승인 완료 후 **"제품 설정" > "카카오맵"** 페이지로 이동
2. **"사용 설정"** 섹션에서
3. **"상태" 토글을 ON으로 변경**
4. **저장**

## ✅ 전체 설정 체크리스트

### 필수 설정 (순서대로):

1. **추가 기능 신청** ✅
   - [ ] "앱 설정" > "앱" > "추가 기능 신청"에서 카카오맵 신청
   - [ ] 승인 대기 및 승인 완료 확인

2. **카카오맵 활성화** ✅
   - [ ] "제품 설정" > "카카오맵"에서 상태를 ON으로 변경

3. **로컬 API 활성화** ✅
   - [ ] "제품 설정"에서 "로컬" 메뉴 찾기
   - [ ] "로컬" 활성화 (주소 변환 기능 사용 시)

4. **플랫폼 설정** ✅
   - [ ] "앱 설정" > "앱" > "플랫폼"에서
   - [ ] Web 플랫폼에 `http://localhost:5173` 등록

## 📝 참고사항

### 비즈앱 전환 필요 여부:
- **일반 개발자도 가능**: 추가 기능 신청만으로 사용 가능
- **비즈앱 전환 불필요**: 개발/테스트 목적이라면 일반 앱으로도 가능
- **상업적 사용**: 대규모 상업적 사용 시 비즈앱 전환 필요할 수 있음

### 승인 시간:
- **일반적으로 빠름**: 몇 분 ~ 몇 시간 내 승인
- **영업일 기준**: 주말/공휴일에는 지연될 수 있음
- **알림 확인**: 승인 완료 시 알림 확인

## 🆘 문제 해결

### "추가 기능 신청" 메뉴를 찾을 수 없는 경우:
- URL로 직접 접속 시도
- 브라우저 새로고침
- 다른 브라우저에서 시도

### 승인이 오래 걸리는 경우:
- "추가 기능 신청" 페이지에서 상태 확인
- 카카오 개발자 센터에 문의 (필요한 경우)

### 승인 후에도 토글이 ON으로 변경되지 않는 경우:
- 페이지 새로고침
- 브라우저 캐시 클리어
- 다른 브라우저에서 시도

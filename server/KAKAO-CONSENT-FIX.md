# 카카오 동의항목 설정 오류 해결

## 문제
"필수 항목을 입력하지 않았거나, 입력한 값이 올바르지 않습니다" 오류가 발생합니다.

## 해결 방법

### 방법 1: 선택 동의로 설정 (가장 간단)

**중요**: 선택 동의로 설정해도 카카오 로그인은 정상 작동합니다!

1. 동의항목 설정 모달에서:
   - "선택 동의" 라디오 버튼 선택
   - 저장

2. 이렇게 해도:
   - 카카오 로그인 시 사용자에게 동의 요청이 나타납니다
   - 사용자가 동의하면 정보를 받을 수 있습니다
   - 동의하지 않아도 로그인은 가능합니다 (다만 정보는 받지 못함)

### 방법 2: 추가 기능 신청 (필수 동의가 꼭 필요한 경우)

만약 반드시 필수 동의가 필요하다면:

1. **추가 기능 신청** 메뉴로 이동
2. "동의항목 설정 권한" 신청
3. 카카오 승인 대기 (보통 1-2일 소요)
4. 승인 후 필수 동의 설정 가능

## 권장: 선택 동의로 진행

**개발 단계에서는 선택 동의로 충분합니다!**

1. 동의항목 설정에서 "선택 동의" 선택
2. 저장
3. Redirect URI 등록
4. REST API 키와 Client Secret 확인
5. `.env` 파일 설정
6. 테스트

## 실제 동작

- 선택 동의로 설정해도:
  - 카카오 로그인 버튼 클릭 → 카카오 로그인 페이지
  - 사용자가 동의하면 정보 수신 가능
  - 동의하지 않아도 로그인은 가능 (정보만 없음)

## 프로덕션 환경

나중에 실제 서비스 배포 시:
- 추가 기능 신청하여 필수 동의 권한 획득
- 또는 선택 동의로 유지 (사용자 선택권 보장)

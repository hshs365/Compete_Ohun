# 테스트 데이터 생성 스크립트

## 사용 방법

### 1. 서버가 실행 중인지 확인
```bash
npm run start:dev
```

### 2. 테스트 모임 데이터 생성
```bash
npm run seed:groups
```

## 생성되는 데이터

- **지역**: 서울, 대전, 부산
- **운동 종목**: 배드민턴, 축구, 농구, 테니스, 등산, 야구, 배구, 탁구, 골프, 클라이밍, 러닝
- **각 지역별 운동 종목당**: 8개씩 생성 (총 약 264개)

## 주의사항

- 스크립트는 `test@example.com` 계정을 사용합니다.
- 계정이 없으면 자동으로 생성됩니다.
- 기존 모임 데이터와 중복될 수 있습니다.

# 카카오맵 스크립트 로드 실패 해결 가이드

## ❌ 현재 발생 중인 에러

```
카카오맵 스크립트 로드 실패: Event {...}
```

이 에러는 카카오맵 JavaScript API 스크립트를 로드하지 못할 때 발생합니다.

## 🔍 원인 확인 방법

### 1단계: Network 탭에서 확인

1. **F12 → Network 탭**
2. **필터에 "kakao" 입력**
3. **`dapi.kakao.com/v2/maps/sdk.js` 요청 확인**
4. **상태 코드 확인**:
   - ✅ **200**: 정상 (다른 문제일 수 있음)
   - ❌ **403**: JavaScript 키가 잘못되었거나 플랫폼 설정 문제
   - ❌ **404**: URL이 잘못됨
   - ❌ **CORS 에러**: 플랫폼 설정에 localhost:5173이 없음

### 2단계: JavaScript 키 확인

1. **카카오 개발자 콘솔 접속**
   - [https://developers.kakao.com/](https://developers.kakao.com/)
   - 애플리케이션 "Ohun" 선택

2. **"앱 키" 탭에서 확인**
   - **JavaScript 키** 복사
   - `.env` 파일의 `VITE_KAKAO_JAVASCRIPT_KEY`와 비교

3. **키가 다른 경우**
   ```env
   # client/.env 파일 수정
   VITE_KAKAO_JAVASCRIPT_KEY=여기에_올바른_JavaScript_키_붙여넣기
   ```

### 3단계: 플랫폼 설정 확인

1. **"플랫폼" 탭 클릭**
2. **Web 플랫폼** 확인:
   - ✅ `http://localhost:5173` 등록되어 있어야 함
   - 없으면 "Web 플랫폼 등록" 클릭 후 추가

3. **저장 후 확인**
   - 변경 사항이 반영되기까지 몇 분 걸릴 수 있음

## 🔧 해결 방법

### 방법 1: JavaScript 키 재확인

1. 카카오 개발자 콘솔 → "앱 키" 탭
2. **JavaScript 키** 복사 (REST API 키 아님!)
3. `.env` 파일에 올바르게 설정
4. 클라이언트 재시작

### 방법 2: 플랫폼 설정 확인

1. 카카오 개발자 콘솔 → "플랫폼" 탭
2. Web 플랫폼에 `http://localhost:5173` 추가
3. 저장
4. 클라이언트 재시작

### 방법 3: 브라우저 캐시 클리어

1. **F12 → Network 탭**
2. **"Disable cache" 체크**
3. **페이지 새로고침 (Ctrl+Shift+R)**

## 🧪 테스트

1. **브라우저 콘솔 확인**:
   - "카카오맵 스크립트 로드 시도" 메시지 확인
   - "카카오맵 스크립트 로드 완료" 메시지 확인

2. **Network 탭 확인**:
   - `dapi.kakao.com/v2/maps/sdk.js` 요청이 **200 OK**인지 확인

3. **지도 표시 확인**:
   - 홈화면에 카카오맵이 표시되는지 확인
   - "새 모임 만들기" → "지도 보기"에서 카카오맵이 표시되는지 확인

## 📝 참고

- **JavaScript 키**와 **REST API 키**는 다릅니다!
- JavaScript 키는 지도 표시용
- REST API 키는 주소 변환용
- 두 키 모두 필요합니다

## 🆘 여전히 문제가 있는 경우

1. **Network 탭에서 실제 에러 메시지 확인**
2. **카카오 개발자 콘솔에서 앱 상태 확인**
3. **JavaScript 키를 다시 발급받아 시도**

# ERR_BLOCKED_BY_CLIENT 에러 해결 가이드

## ❌ 현재 발생 중인 에러

```
Status: (failed) net::ERR_BLOCKED_BY_CLIENT
```

이 에러는 **브라우저 확장 프로그램**(주로 광고 차단기)이 카카오맵 스크립트를 차단하고 있다는 의미입니다.

## 🔧 해결 방법

### 방법 1: 광고 차단기 비활성화 (가장 빠른 방법)

1. **브라우저 확장 프로그램 아이콘 클릭**
   - 주소창 오른쪽의 확장 프로그램 아이콘 (퍼즐 모양 또는 차단 아이콘)
   - 또는 `chrome://extensions/` 접속

2. **광고 차단기 찾기**
   - AdBlock, uBlock Origin, AdGuard 등
   - 또는 기타 차단 확장 프로그램

3. **일시적으로 비활성화**
   - 확장 프로그램 토글을 OFF로 변경
   - 또는 `localhost:5173`에 대해 예외 추가

4. **페이지 새로고침** (Ctrl+R)

### 방법 2: 시크릿 모드에서 테스트

1. **시크릿 모드 열기**
   - Ctrl+Shift+N (Chrome)
   - 또는 Ctrl+Shift+P (Firefox)

2. **`http://localhost:5173` 접속**

3. **카카오맵이 작동하는지 확인**
   - 작동한다면 확장 프로그램 문제 확정

### 방법 3: 확장 프로그램 예외 추가

1. **광고 차단기 설정 열기**
   - 확장 프로그램 아이콘 클릭
   - "옵션" 또는 "설정" 클릭

2. **예외 사이트 추가**
   - `localhost:5173` 추가
   - 또는 `dapi.kakao.com` 추가

3. **페이지 새로고침**

### 방법 4: 모든 확장 프로그램 비활성화

1. **확장 프로그램 관리 페이지 접속**
   - `chrome://extensions/` (Chrome)
   - 또는 `edge://extensions/` (Edge)

2. **모든 확장 프로그램 일시적으로 비활성화**
   - 각 확장 프로그램의 토글을 OFF로 변경

3. **페이지 새로고침**

4. **카카오맵이 작동하는지 확인**
   - 작동한다면 확장 프로그램 문제 확정
   - 어떤 확장 프로그램이 문제인지 하나씩 활성화하며 확인

## ✅ 빠른 해결 방법 (권장)

### 1단계: 시크릿 모드에서 테스트
1. **Ctrl+Shift+N** (시크릿 모드 열기)
2. **`http://localhost:5173` 접속**
3. **카카오맵이 작동하는지 확인**

### 2단계: 작동한다면 확장 프로그램 문제
1. **일반 모드로 돌아가기**
2. **광고 차단기 비활성화**
3. **또는 `localhost:5173`에 대해 예외 추가**

## 🧪 테스트

### 시크릿 모드에서:
1. **Ctrl+Shift+N** (시크릿 모드)
2. **`http://localhost:5173` 접속**
3. **F12 → Network 탭**
4. **필터에 "kakao" 입력**
5. **`sdk.js` 요청 확인**
6. **상태 코드가 200 OK인지 확인**

### 일반 모드에서 (확장 프로그램 비활성화 후):
1. **광고 차단기 비활성화**
2. **페이지 새로고침** (Ctrl+R)
3. **F12 → Network 탭**
4. **필터에 "kakao" 입력**
5. **`sdk.js` 요청 확인**
6. **상태 코드가 200 OK인지 확인**

## 📝 참고

- **ERR_BLOCKED_BY_CLIENT**는 카카오 개발자 콘솔 설정 문제가 아닙니다
- **브라우저 확장 프로그램**이 요청을 차단하고 있습니다
- **시크릿 모드**에서는 확장 프로그램이 기본적으로 비활성화되어 있어서 테스트하기 좋습니다

## 🆘 여전히 문제가 있는 경우

시크릿 모드에서도 작동하지 않는다면:
1. **다른 브라우저에서 테스트** (Chrome → Edge 또는 Firefox)
2. **카카오 개발자 콘솔 설정 다시 확인**
   - JavaScript SDK 도메인 등록 확인
   - 추가 기능 신청 확인
   - 카카오맵 활성화 확인

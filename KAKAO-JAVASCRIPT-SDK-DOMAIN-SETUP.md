# 카카오맵 JavaScript SDK 도메인 등록 가이드

## ⚠️ 중요: 올바른 설정 위치

현재 보고 계신 "제품 링크 관리" 페이지는 **카카오톡 공유용** 도메인 설정입니다.

카카오맵 JavaScript API를 사용하려면 **다른 곳**에서 설정해야 합니다!

## ✅ 올바른 설정 위치

### 사이드바 경로:
```
앱 설정
  └─ 앱
      └─ 플랫폼 키 (Platform Key)
          └─ JavaScript 키
              └─ JavaScript SDK 도메인
```

**단계별 안내:**

1. **왼쪽 사이드바에서 "앱 설정" 클릭**
2. **"앱" 섹션 확장** (화살표 클릭)
3. **"플랫폼 키" 메뉴 클릭**
4. **"JavaScript 키" 섹션 찾기**
5. **"JavaScript SDK 도메인" 섹션 찾기**
6. **"도메인 추가" 또는 "수정" 버튼 클릭**
7. **`http://localhost:5173` 입력**
8. **저장**

### 또는 URL로 직접 접속:
```
https://developers.kakao.com/console/app/1358790/config/platform-key
```

## 📝 두 가지 도메인 설정의 차이

### 1. 제품 링크 관리 (현재 보고 계신 페이지)
- **용도**: 카카오톡 공유, 카카오톡 메시지 링크 이동
- **위치**: "앱 설정" > "앱" > "제품 링크 관리"
- **카카오맵과의 관계**: ❌ 카카오맵 JavaScript API와는 무관

### 2. JavaScript SDK 도메인 (필수!)
- **용도**: Kakao SDK for JavaScript 사용 (카카오맵 포함)
- **위치**: "앱 설정" > "앱" > "플랫폼 키" > "JavaScript 키" > "JavaScript SDK 도메인"
- **카카오맵과의 관계**: ✅ **카카오맵 JavaScript API 사용에 필수!**

## 🔧 JavaScript SDK 도메인 등록 방법

### 1단계: 플랫폼 키 페이지 접속

1. **"앱 설정" > "앱" > "플랫폼 키" 클릭**
2. 또는 URL로 직접 접속:
   ```
   https://developers.kakao.com/console/app/1358790/config/platform-key
   ```

### 2단계: JavaScript 키 섹션 찾기

1. **"JavaScript 키" 섹션 찾기**
   - "대표 Default JS Key" 카드 확인
   - 키 값: `e808ef5da75ce1c3d2f548a9794935a0`

### 3단계: JavaScript SDK 도메인 등록

1. **"JavaScript SDK 도메인" 섹션 찾기**
   - JavaScript 키 카드 아래에 있을 수 있음
   - 또는 별도 섹션으로 표시될 수 있음

2. **"도메인 추가" 또는 "수정" 버튼 클릭**

3. **도메인 입력**:
   ```
   http://localhost:5173
   ```

4. **저장**

## ✅ 전체 설정 체크리스트

### 필수 설정 (순서대로):

1. **추가 기능 신청** ✅
   - [ ] "앱 설정" > "앱" > "추가 기능 신청"에서 카카오맵 신청
   - [ ] 승인 대기 및 승인 완료 확인

2. **카카오맵 활성화** ✅
   - [ ] "제품 설정" > "카카오맵"에서 상태를 ON으로 변경

3. **JavaScript SDK 도메인 등록** ✅ (현재 단계)
   - [ ] "앱 설정" > "앱" > "플랫폼 키" > "JavaScript 키" > "JavaScript SDK 도메인"
   - [ ] `http://localhost:5173` 등록

4. **로컬 API 활성화** (선택사항)
   - [ ] "제품 설정"에서 "로컬" 활성화 (주소 변환 기능 사용 시)

## 🆘 JavaScript SDK 도메인 섹션을 찾을 수 없는 경우

1. **플랫폼 키 페이지에서 JavaScript 키 카드 클릭**
   - 카드를 확장하면 SDK 도메인 설정이 보일 수 있음

2. **JavaScript 키 옆의 "수정" 또는 "설정" 버튼 클릭**
   - 별도 모달이나 페이지가 열릴 수 있음

3. **페이지 새로고침**
   - 브라우저 캐시 문제일 수 있음

4. **카카오 개발자 센터에 문의**
   - UI가 변경되었을 수 있음

## 📝 참고

- **제품 링크 관리**에 등록한 도메인은 카카오맵과 무관합니다
- **JavaScript SDK 도메인**에 등록해야 카카오맵이 작동합니다
- 두 가지는 별도로 관리됩니다
